# Suricata

Landing page para tienda de alimentos frescos y preparados.

## Tecnologías

- HTML5
- CSS3
- Bootstrap 5.3
- Bootstrap Icons 1.11
- Payment Icons (SVG logos de medios de pago)

## Secciones

- **Navbar** responsive con menú hamburguesa
- **Hero** con llamado a la acción
- **Productos** en grilla de cards (colaciones, desayunos, bebidas, verduras, dulces, empanadas)
- **Medios de pago** con logos oficiales (Visa, Mastercard, Amex, Diners, PayPal, Discover)
- **Contacto** con WhatsApp, Instagram, ubicación y horario
- **Footer**

## Vista previa

Abrir `index.html` directamente en el navegador.
